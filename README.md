# Kamelrechner
Der Kamelrechner ist eine humorvolle Webanwendung, die anhand verschiedener Eingaben wie Alter, Hobbys, Bildungsstand und Gewohnheiten den "Marktwert" einer Person in Kamelen, Ochsen, Eseln, Schafen, Ziegen und Schweinen berechnet.
